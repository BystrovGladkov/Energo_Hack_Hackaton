Для запуска проекта сначала необходимо настроить окружение python:

Команды на Linux:

`python -m venv venv`

`source ./venv/bin/activate`

`python -m pip install -r requirements.txt`

Команды для Windows:

`python -m venv venv`

`venv\Scripts\activate.bat`

`python -m pip install -r requirements.txt`
